## Briefing Executivo

### O Desafio Estratégico

Criar um portfólio vivo que demonstre, de forma tangível, a capacidade de automação e inteligência artificial do Encontro d'Água Hub. A **Amazô** não é apenas um chatbot tradicional; é uma **SDR digital** que:

- ✅ Qualifica leads de forma inteligente e contextual
- ✅ Apresenta os serviços do Hub de maneira humana e acessível
- ✅ Captura dados estratégicos do usuário
- ✅ Converte interessados em clientes através de um fluxo de venda humanizado
- ✅ Demonstra a viabilidade de IA acessível para pequenos negócios (a partir de R$ 80,00)

### Objetivo Principal

Democratizar a tecnologia de IA, construindo agentes sob medida para pequenos negócios e projetos sociais, utilizando metodologia heutagógica e pair programming com IA.

## Recursos e Funcionalidades

### Core Features

| Recurso | Descrição |
| --- | --- |
| **Integração Humano-IA** | Transição suave da conversa com o assistente virtual para a equipe humana no WhatsApp, sem atritos |
| **Compreensão Contextual** | Agentes que fogem de fluxos engessados, entendendo o contexto real e adaptando as respostas |
| **Modo Concierge** | Identidade, regras de negócio e tom de voz configurados 100% para refletir sua marca |
| **Chatbot Integrado (Typebot)** | Modal interativo com conversação via IA, acessível direto na página |
| **Recepção e SAC 24/7** | A IA tira dúvidas frequentes, acolhe leads e informa horários/links a qualquer momento |
| **Otimização de Vendas (SDR)** | A IA atua como representante de vendas, qualificando o lead antes da passagem para fechamento |
| **Design Tech Amazônico** | Interface dark mode com paleta Rio Negro, Açaí Neon e Dourado Solimões |
